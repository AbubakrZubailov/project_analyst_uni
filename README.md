# project_analyst_uni
Описание проекта

Проект посвящён анализу и проектированию системы управления доступом и учётными записями вуза (IAM).

Система предназначена для централизованного управления пользователями, их ролями, правами доступа и учётными записями в информационных ресурсах университета. Основными пользователями системы являются студенты, преподаватели, сотрудники и администраторы.

В рамках проекта были разработаны следующие модели и диаграммы:
1. BPMN-диаграмма бизнес-процесса управления доступом;
2. Use Case Diagram для описания взаимодействия пользователей с системой;
3. ER Diagram для проектирования структуры базы данных;
4. Sequence Diagram для отображения последовательности взаимодействий между компонентами системы;
5. Class Diagram для описания архитектуры предметной области.

Студенческий проект по дисциплине «Моделирование бизнес-процессов».
